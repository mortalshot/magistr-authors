Расчёт рейтинга

Для каждого автора на основе его последних 30 отзывов рассчитывается балл.

Формула: процент положительных отзывов + процент нейтральных / 2

Далее авторы сортируются по убыванию. Вверху рейтинга авторы с самыми хорошими отзывами. Затем полученный список делим на 5 групп с шагом в 20% Верхние 20% назовем их группой с рейтингом 5 (отличники), соответственно следующие 20% это группа с рейтингом 4 (хорошие авторы). Авторы с самыми плохими отзывами получат рейтинг 1.

Пример расчета баллов:

Пример 1
Рейтинг в процентах
60%
34%
6%
Рейтинг в баллах
60 + 34 / 2 = 77
Пример 2
Рейтинг в процентах
61%
6%
33%
Рейтинг в баллах
61 + 6 / 2 = 64

Влияние на базовую стоимость для автора

Для каждой группы установить свою базовую стоимость

5 группа (отличный)
+ 10% к БЗ
4 группа (хороший)
+ 5% к БС
3 группа (троечники)
0% к БС
2 группа (двоечники)
- 5% к БС
1 группа (черный список)
- 10% к БС

Примечания

  1. БС - базовая стоимость
  2. Значения процентов ориентировочные, планируется плавное увеличение, чтобы дать авторам возможность улучшить свои показатели
  3. Через некоторое время, для определения групп рейтинга возможен переход к фиксированному числовому диапазону
  4. Для определения рейтинга целенаправленно выбрано малое число отзывов, чтобы автор быстрее смог улучшить свои показатели
  5. Рейтинг обновляется один раз в день
